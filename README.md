# Belora Mobility — Website

Premium marketing site for Belora Mobility: professionally operated on-site bike fleets for large hospitality resorts and industrial sites.

Static site (HTML / CSS / vanilla JS) with a Three.js soft-isometric hero, GSAP + ScrollTrigger choreography, and Lenis smooth scroll. Brand fonts: Clash Display (display) + Inter (body).

## Run locally
Any static server, e.g.:

```bash
python3 -m http.server 4321
```

then open http://localhost:4321

## Structure
- `index.html` — markup
- `css/styles.css` — design system (CSS variables) + all styles
- `js/main.js` — Three.js hero, animations, interactions
- `assets/` — logos, optimized imagery, testimonial video
